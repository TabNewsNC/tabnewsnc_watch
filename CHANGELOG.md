# Changelog

[TabNews](https://tabnews.com.br) client for smartwatch built in
[Flutter](https://flutter.dev) to view and interact with news and articles
published on the platform.

## 0.1.0

* Packages
  * Added the `wear_plus` package
  * Added the `tabnewsnc_ui` package
  * Added the `flutter_modular` package
  * Added the `tabnewsnc_engine` package
  * Added the `rotary_scrollbar` package

* Theme
  * Added theme from `tabnewsnc_ui` package

* Widgets
  * Added the `Scrollbar` widget

* Screen
  * Added the `Root` screen
  * Added the `Home` screen

* Others
  * Added the splash screen
  * Added the launcher icons

* Features
  * Listing of news and articles by relevance
